npm / bunx

If you prefer Node.js package managers. Auto-bootstraps Python on first run.

# npx (auto-installs Python core)
npx @nusabyte/jebat repl

# bunx
bunx @nusabyte/jebat repl

# Or install globally
npm install -g @nusabyte/jebat
jebat repl

IDE Integration

Connect JEBAT to your favorite IDE via MCP.

# Generate IDE configs after install
jebat mcp --print-config

# Start MCP server (HTTP mode)
jebat mcp serve --port 8206

# Start MCP server (stdio mode — for IDE integration)
jebat mcp serve

VS Code / Cursor

// .vscode/settings.json or Cursor MCP settings
{
  "mcpServers": {
    "jebat": {
      "command": "jebat",
      "args": ["mcp", "serve"]
    }
  }
}
