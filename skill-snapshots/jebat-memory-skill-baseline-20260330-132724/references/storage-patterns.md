# Memory Storage Patterns

## Daily memory
Use `memory/YYYY-MM-DD.md` for:
- events
- decisions-in-progress
- short-term task context

## Long-term memory
Use `MEMORY.md` for:
- preferences
- stable project context
- important architecture decisions
- enduring lessons

## Procedural memory
Use:
- `AGENTS.md`
- `TOOLS.md`
- skill docs
- dedicated procedure docs

## Rule
If Jebat should behave differently in the future because of a discovery, write it down in a durable file.
