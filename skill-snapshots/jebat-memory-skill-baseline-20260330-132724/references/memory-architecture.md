# Memory Architecture

## Layers
- M0 sensory: transient working context
- M1 episodic: daily events and interactions
- M2 semantic: facts, preferences, relationships
- M3 conceptual: durable principles and mental models
- M4 procedural: workflows, habits, operating patterns

## Promotion guidance
- event worth remembering today -> M1
- repeated fact or preference -> M2
- durable architecture principle -> M3
- repeatable successful workflow -> M4

## Heat model
Approximate importance using:
- repeated recall
- depth of interaction
- recency
- cross-links
- explicit importance

Use this conceptually inside OpenClaw even when the full database-backed model is not implemented.
