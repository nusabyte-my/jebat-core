# Research Patterns

## Workflow
1. search
2. fetch best sources
3. compare consensus and disagreement
4. synthesize
5. cite
6. store important findings if they matter later

## Advanced academic mode
Use when the user needs:
- scholarly papers by topic, author, or DOI
- citation-chain exploration
- structured literature review
- curated paper-set building

Patterns worth using:
- general academic lookup first
- advanced model-led query expansion only when deeper collection quality matters
- dedupe and triage before synthesis

## Output pattern
Return:
- concise answer
- strongest sources
- disagreements if relevant
- confidence / uncertainty
- for academic tasks: paper metadata, relevance, and next reading suggestions

## Rule
Synthesize. Do not just paste search results.
