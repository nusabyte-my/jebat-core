# JEBAT Hardening Skill

Use this skill to **turn security findings into safer configuration and deployment state**.

## Core workflow
1. inspect current state
2. identify weak defaults, exposure, and missing controls
3. propose minimal safe improvements first
4. back up or plan reversibility before risky changes
5. verify results after change

## Read references when needed
- `references/hardening-patterns.md` — standard hardening workflow and focus areas

## Operational rules
- ask before disruptive production changes
- prefer reversible edits
- tie every hardening change to a clear risk reduction
