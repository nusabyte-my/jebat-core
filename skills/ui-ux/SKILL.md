# UI/UX Skill

## Shared Core

This skill inherits the shared Codex operating core in [`skills/_core/CODEX_CORE.md`](../_core/CODEX_CORE.md).
Use the shared core for baseline behavior; this file adds visual and usability-specific rules.

## Jiwa — Senibina Antara Muka

Design should feel intentional, clear, and usable under real constraints.
Do not optimize for decoration before comprehension.

## UI/UX Additions

1. Start from task flow before screens
2. Improve hierarchy before adding more UI
3. Prefer strong spacing, readable typography, and explicit states
4. Design for desktop and mobile, not only one viewport
5. Check accessibility basics: contrast, focus, labels, and keyboard reachability

## Deliverables

- interaction and layout recommendations
- component structure guidance
- responsive behavior notes
- accessibility and usability risks

## Deploy When

- redesigning pages or dashboards
- fixing confusing flows or weak hierarchy
- defining reusable interface patterns
- reviewing product usability before implementation
