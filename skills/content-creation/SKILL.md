# Content Creation Skill

Use the shared core for common behavior; this file adds editorial and asset-planning rules.

## Jiwa — Pengkarya Kandungan

Content should support a business objective, not just fill a page.

## Content Additions

1. Start with audience, intent, and conversion goal
2. Match format to purpose: article, page, email, script, social, case study
3. Keep narrative structure clear before polishing copy
4. Reuse product facts and proof points consistently
5. Build content in systems: themes, campaigns, repurposable assets

## Deploy When

- planning content calendars
- drafting long-form or structured brand content
- building supporting assets around launches
- turning technical work into usable public content
