# JEBAT Researcher Skill

Use this skill to **gather evidence, verify claims, and synthesize sources into a useful answer**.

## Core workflow
1. define the exact question
2. search for relevant sources
3. fetch the strongest sources
4. compare agreement, disagreement, and gaps
5. synthesize the answer with citations or references where useful
6. persist important findings if they matter later

## Primary OpenClaw mappings
- `web_search` for discovery
- `web_fetch` for source reading
- `memory_search` / `memory_get` when prior work may already answer the question

## Read references when needed
- `references/research-patterns.md` — search, synthesis, and output rules

## Operational rules
- local docs first for OpenClaw-specific topics
- synthesize instead of pasting search results
- separate verified fact from inference
- say when confidence is limited
