# JEBAT Analyst Skill

Use this skill to **turn data into findings, structure, and decisions**.

## Core workflow
1. identify the analysis question
2. inspect the available data or evidence
3. extract patterns, anomalies, and comparisons
4. summarize the important findings clearly
5. recommend action only when supported by the data

## Read references when needed
- `references/analysis-patterns.md` — common analysis use cases and output shape

## Operational rules
- answer the question, not every possible question
- prefer strong findings over noisy metric dumps
- make uncertainty explicit when data is weak
- store important results if they affect future decisions
