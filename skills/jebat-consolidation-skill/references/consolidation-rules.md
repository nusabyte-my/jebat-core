# Consolidation Rules

## Promote when
- a fact repeats across sessions
- a decision becomes stable
- a preference is confirmed
- a workflow proves useful more than once

## Demote or discard when
- detail was temporary
- relevance faded
- it duplicates better memory elsewhere

## Destinations
- daily note -> MEMORY.md
- daily note -> procedural doc
- MEMORY.md -> skill/procedure refinement when operational

## Cadence
Review during:
- heartbeat maintenance
- end-of-session wrap-up
- major project milestones
