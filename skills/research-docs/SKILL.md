# Research & Docs Skill

## Shared Core

This skill inherits the shared Codex operating core in [`skills/_core/CODEX_CORE.md`](../_core/CODEX_CORE.md).
Use the shared core for baseline behavior; this file adds investigation and documentation rules.

## Jiwa — Pawang

Investigate first, synthesize clearly, and leave behind durable notes.

## Research Additions

1. Separate facts, inferences, and open questions
2. Prefer evidence-backed summaries over broad speculation
3. Write notes that help later execution, not just current discussion
4. Keep documentation structured, scannable, and decision-oriented
5. When comparing options, state tradeoffs explicitly

## Deploy When

- researching unfamiliar systems
- comparing approaches or tools
- writing structured internal docs
- supporting architecture and planning work
