# Session End Procedure

## Goal

Make session closure consistent and useful.

---

## Routine
1. run wrap-up scaffold if needed
2. record decisions and follow-ups in daily memory
3. promote durable items into `MEMORY.md`
4. update procedures if a reusable workflow was learned
5. leave unresolved work clearly stated
