## 🎯 How to Push to GitHub

### Option 1: Install Git and Use Script (Recommended)

#### Step 1: Install Git

1. Download from: https://git-scm.com/
2. Run installer
3. Use default settings
4. Finish installation
5. **Restart your terminal/command prompt**

#### Step 2: Run Push Script

```bash
# After installing git, run:
git_push.bat
```

The script will:
1. Initialize git repository
2. Stage all files
3. Commit with message
4. Guide you to add remote
5. Push to GitHub

#### Step 1: Install Git

Download and install from: https://git-scm.com/

1. Go to https://github.com
2. Click "+" → "New repository"
3. Name: `jebat-core`

Open Command Prompt in `C:\Users\shaid\Desktop\Dev`:

## 🐛 Troubleshooting

### "Git is not recognized"

**Solution:** Install git from https://git-scm.com/ and restart terminal

### "Permission denied"

**Solution:** Use personal access token from GitHub:
1. GitHub Settings → Developer settings → Personal access tokens
2. Generate token with `repo` scope
3. Use token as password when pushing

### "Remote origin already exists"

**Solution:**
```bash
git remote remove origin
git remote add origin YOUR_URL
```
