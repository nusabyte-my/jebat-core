## 💡 Usage Examples

### CLI Commands

```bash
# Show system status
py -m jebat.cli.launch status

# Start Ultra-Loop
py -m jebat.cli.launch loop start

# Stop Ultra-Loop
py -m jebat.cli.launch loop stop

# Run thinking session
py -m jebat.cli.launch think "What is AI?"

# Run thinking with specific mode
py -m jebat.cli.launch think "What is the meaning of life?" --mode deep

# Store a memory
py -m jebat.cli.launch memory store "I prefer Python for development"

# Search memories
py -m jebat.cli.launch memory search "Python development"

# Show configuration
```
