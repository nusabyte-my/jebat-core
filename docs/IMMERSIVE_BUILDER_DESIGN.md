## 🎨 Design Philosophy

### **Core Principles**

1. **Invisible Interface**
   - Tools appear when needed
   - Disappear when not
   - Focus on creation, not UI

2. **AI-First, Not AI-Only**
   - AI assists, doesn't replace
   - User always in control
   - Suggestions, not commands

3. **Progressive Complexity**
   - Simple by default
   - Advanced when needed
   - Never overwhelming

4. **Delightful Micro-interactions**
   - Smooth animations
   - Satisfying feedback
   - Playful but professional

5. **Spatial Design**
   - Depth and layers
   - Natural z-axis
   - 3D thinking

/* Dark Theme (Default) */
