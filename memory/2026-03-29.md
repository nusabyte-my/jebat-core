## JEBAT Core Architecture (From Repo)

### Memory System
- **5 Layers:** M0 (sensory, 0-30s) → M1 (episodic, hours) → M2 (semantic, 30d) → M3 (conceptual, permanent) → M4 (procedural, skills)
- **Heat Scoring:** Visit frequency (30%) + interaction depth (25%) + recency (25%) + cross-references (15%) + explicit rating (5%)
- **Automatic Consolidation:** Episodic → Semantic → Conceptual based on heat

### Skill Framework
- Base: `BaseSkill` abstract class
- Skill Types: DATA_PROCESSING, COMMUNICATION, FILE_OPERATIONS, WEB_SCRAPING, API_INTEGRATION, ANALYSIS, AUTOMATION, DATABASE, MACHINE_LEARNING, UTILITY
- Features: Parameter validation, error handling, metrics tracking, dependencies
- Example Skills: `advanced_sql_skill`, `api_communication_skill`, `expert_automation_skill`, `expert_config_skill`, `file_processing_skill`, `fullstack_development_skill`

### Multi-Agent Orchestration
- Core, Memory, Tool, Execution, Analyst, Researcher agents
- Agent factory and orchestrator
- Session management
- Cross-agent communication

### Gateway Architecture (Based on OpenClaw)
- Multi-channel: WhatsApp, Telegram, Discord, Slack, Signal, iMessage, Teams, Matrix, WebChat, Voice
- WebSocket hub on port 18789
- DM pairing for security
- Real-time events and presence
