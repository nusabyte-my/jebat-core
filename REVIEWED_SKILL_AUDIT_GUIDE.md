# Reviewed Skill Audit Guide

## Goal

Summarize the reviewed external skill registry quickly.

---

## Use the helper
```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\reviewed-skill-audit.ps1
```

## Output includes
- total reviewed
- verdict counts
- average score
- raw reviewed skill records

## Use when
- before importing a new skill
- during periodic stack review
- before productionizing a bigger Jebat skill set
