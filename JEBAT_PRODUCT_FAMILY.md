## 🚀 Usage

### Launch Product Selector

```bash
# Run selector
python jebat_selector.py

# Or directly launch a product
python -m jebat_dev.gateway          # JEBAT Dev
python -m jebat.core                 # JEBAT Core (coming)
python -m jebat_security.cli         # JEBAT Security (coming)
python -m jebat_companion.cli        # JEBAT Companion (planned)
python -m jebat_nexus.bot            # JEBAT Nexus (planned)
```


## 🎯 Naming Philosophy

### Malay Heritage
All codenames honor Malay language and culture:
- **Hang Jebat** - The legendary warrior (core platform)
- **Keris** - Traditional dagger (security/precision)
- **Pandai** - Clever/skilled (development expertise)
- **Sahabat** - Friend/companion (chat assistant)
- **Perisai** - Shield (protection/connection)

### Why This Matters
- Reflects JEBAT's cultural roots
- Creates unique brand identity
- Honors the Hang Jebat legend
- Distinguishes from Western AI products
