Core Capabilities

Built-in intelligence systems

Beyond the five products, JEBAT ships with embedded subsystems that make it a true agent operating system.

Ghost DB

Observability Stack

Catalyst O11y

OTel-compatible distributed tracing, Prometheus metrics, Loki logging. HALO analysis engine with Webhook, Slack, PagerDuty alerting.

🔐
Access Control

Enterprise RBAC

Hierarchical org/team/project roles. 10 resource types × actions. FastAPI routes with permission resolution engine.

🐍
Python + TypeScript

SDK

Sync/async Python clients with Pydantic. TypeScript with Zod validation, React hooks, testing mocks. Full API coverage.

Dream Engine

Mimpi

8 dream types with concept graph seeding. Induction → Exploration → Synthesis → Insight → Consolidation → Awakening phases.

Adaptive Intelligence

Self-Learning

MetaLearner with UCB1 strategy selection. Builtin strategies auto-registered. Performance tracking with forgetting curve.

Memory Architecture

6 memory types with Ebbinghaus forgetting curves. Agent loop auto-encodes outcomes. Memory recall injected into every reasoning cycle.

{
  "mcpServers": {
    "jebat": {
      "command": "python",
      "args": ["-m", "jebat.features.mcp.mcp_server"]

  }

HTTP Remote Access

# Start HTTP transport
jebat mcp serve --transport http --port 8099

# Streamable HTTP (MCP 2025-03-26)
jebat mcp serve --transport streamable-http --port 8100

# List available tools
jebat mcp serve --help

CLI Quick Start

# Start as stdio server (for IDE)
jebat mcp serve --transport stdio

# Generate IDE config templates
jebat mcp serve --print-ide-config

npx Zero-Install

# Via npx (no install required)
npx @nusabyte/jebat mcp serve

# Connect external MCP server
jebat mcp connect --http http://host:8100/mcp
