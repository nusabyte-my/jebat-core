Embedded Pentest
Orchestration

Six industry-standard tools orchestrated through a single CLI entrypoint. Automated reconnaissance, route discovery, injection testing, and report generation.

// toolchain

nuclei

Template-based vulnerability scanning. 8000+ community templates for CVE, misconfig, exposure detection.

nmap

Port scanning, service fingerprinting, OS detection. Full NSE script integration.

gobuster

Directory/file brute-forcing, DNS subdomain enumeration, vhost discovery.

sqlmap

Automated SQL injection detection and exploitation. Full POST/GET/cookie injection surface.

nikto

Web server scanner. 6700+ potentially dangerous files/CGIs, outdated version detection.

dnsx + httpx

DNS resolution and HTTP probing. Subdomain validation and live host filtering.

// pipeline
# Reconnaissance Phase
$ jebat pentest -t example.com --recon
→ nuclei: 12 findings · nmap: 8 open ports · gobuster: 24 dirs
# Injection Phase
$ jebat pentest -t example.com --inject
→ sqlmap: 3 injectable params · nikto: 2 CVE matches
# Full Spectrum
$ jebat pentest -t example.com --full
→ Report: ~/reports/pentest_example.com_20260331.md

Multi-Agent Swarm
Orchestration

Pasukan JEBAT — specialized role-based agents dispatched in parallel. Each swarm role carries a distinct behavioral profile, cost budget, and model preference.

// swarm roles

Planner

Breaks goals into actionable tasks. Strategic decomposition with dependency ordering.

Implementer

Executes planned tasks — code generation, file manipulation, shell commands.

Reviewer

Validates output against requirements. Security scan, lint, style checks.

Researcher

Web search, documentation lookup, API discovery. Context gathering.

Critic

Adversarial testing. Finds edge cases, failure modes, and security holes.

Synthesizer

Merges swarm outputs. Conflict resolution via Taming Sari consensus algorithm.

// features
◈

Role-Based Profiles

Each swarm role uses its own cavement/lean/deep profile. Planner uses deep, implementer uses lean, reviewer uses cavement — cost-optimized by role.

◈

Swarm Result Cache

SHA256-hashed goals skip re-dispatch. Cache stats via `jebat cache stats`. Invalidate with `jebat cache invalidate --age 24h`.

◈

Taming Sari Consensus

LEGENDARY mode only. Detects contradictions between role outputs, marks consensus/veto per claim, merges non-conflicting sections.

◈

AgentLoop Bridge

--execute flag runs swarm roles via AgentLoop, not just prints prompts. Real LLM dispatch with failover.
