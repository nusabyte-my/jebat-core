Quick Start

Start in 30 seconds

npx Zero Install

# Interactive REPL
npx @nusabyte/jebat repl

# One-shot chat
npx @nusabyte/jebat chat "Explain memory consolidation"

# With bun
bunx @nusabyte/jebat repl

pip Global Install

# Via pip
pip install jebat
jebat repl

# Via npm global
npm install -g @nusabyte/jebat
jebat repl

MCP IDE Integration

# Add to .cursor/mcp.json
{
  "mcpServers": {
    "jebat": {
      "command": "python",
      "args": ["-m", "jebat.mcp.server"]
    }
  }
}

git Source Build

# Clone & install
git clone https://github.com/nusabyte-my/jebat-core.git
cd jebat-core
pip install -e .
jebat repl
