# Runtime Logs Index

## Operational logs
- `DECISION_PROVENANCE_LOG.md` — major decisions and why they were made
- `LIFECYCLE_NOTES.md` — change history and milestone transitions
- `task-records.md` — task records created during runtime execution
- `security-remediation-log.md` — remediation entries for security/runtime gaps
- `OPS_DASHBOARD.generated.md` — generated operational status snapshot
- `BRAIN_STATUS.generated.md` — generated brain/entity memory status snapshot

## Memory logs
- `memory/YYYY-MM-DD.md` — daily working memory
- `MEMORY.md` — long-term distilled memory
- `brain/INDEX.md` — entity memory index

## Usage
Use this file as the quick pointer map when checking Jebat's live operational state.
