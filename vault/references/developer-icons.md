# Developer Icons Reference
# Source: https://github.com/xandemon/developer-icons
# Usage: Use these icon identifiers in UI/UX agent output for tech stack visualization.

## How to Use in Agent Output
When generating UI mockups, landing pages, or documentation, reference icons using the format:
- `icon:React` → React logo
- `icon:Node.js` → Node.js logo
- `icon:TailwindCSS` → Tailwind CSS logo

Icons are SVG-based, fully scalable, and support color customization via standard SVG fill/stroke.

## Icon Usage Guidelines
- Always reference by icon name, not SVG filename, in agent output
- Icons should be displayed at 16px-48px depending on context
- Use monochrome (currentColor) for inline text integration
- Use full-color for tech stack grids and landing page sections
- Maintain consistent sizing within each usage context
- Dark/light variants available where noted (Express, Flask, Apple, Anthropic, Framer)
