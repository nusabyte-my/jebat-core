# Renewal Brief Example

## Client / Engagement
Operations dashboard retainer approaching annual renewal.

## Current Scope
- monthly improvements
- support and onboarding refinement
- reporting adjustments
- lightweight automation

## Outcomes Delivered
- improved ownership tracking
- faster reporting cadence
- lower repeated admin confusion

## Performance Signals
- adoption is improving
- request volume for new features is growing
- dashboard is becoming central to daily operations

## Support / Relationship Signals
- client satisfaction is strong
- support requests are more advanced now, indicating maturity rather than confusion

## Scope Drift / Change Requests
- external reporting visibility
- more advanced permissions
- deeper workflow automation

## Renewal Recommendation
Renew with a higher tier that preserves support cadence while adding a scoped roadmap lane for higher-impact improvements.

## Expansion / Reduction Options
- expand into reporting and permissions roadmap
- keep current support lane and add quarterly roadmap planning
- defer major integrations into separate scoped proposals

## Risks / Dependencies
- expansion without clearer boundaries will create delivery risk
- client expectations need to be reset around what stays in retainer versus project scope

## Next Step
Run `renewal-strategy` and prepare a renewal recommendation plus follow-up proposal for expanded roadmap work.

## Roles Needed
- Penulis Cadangan
- Khidmat Pelanggan
- Penganalisis
- Strategi Produk
