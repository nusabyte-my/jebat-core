# SEO Verification Checklist

Use for work led by Penjejak Carian or any task that aims to improve discoverability.

## Before Sign-Off

- Search intent is explicitly named
- Title, headings, and body align to the same intent
- Metadata needs are covered
- Internal linking and information architecture were considered
- Content avoids thin, duplicate, or misaligned messaging

## Verification

- On-page structure was checked
- Metadata changes are documented
- Internal-link opportunities or gaps are noted
- Residual SEO risks or dependencies are listed
